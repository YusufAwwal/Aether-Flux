# Aether Flux: Web3 Command Center Roadmap

Aether Flux is a premium, real-time blockchain visualizer and portfolio hub designed with a high-fidelity "Cyber-Premium" aesthetic.

## Tech Stack
- **Frontend**: Next.js (App Router), React, Lucide Icons.
- **Styling**: Vanilla CSS (CSS Modules/Variables) for precision.
- **Web3**: ethers.js / wagmi / viem.
- **Animations**: Framer Motion / CSS Transitions.

## 100 Commit Strategy

### Phase 1: Infrastructure & Design System (Commits 1-15)
- **1-3**: Project initialization (Next.js, Folder structure).
- **4-7**: Design Tokens (Colors, Typography, Spacing).
- **8-12**: Core UI Elements (Buttons, Glass Containers, Inputs).
- **13-15**: Layout Shell (HUD-style Sidebar and Navigation).

### Phase 2: Dashboard Core (Commits 16-40)
- **16-25**: Flux Stream (Real-time TX list with micro-animations).
- **26-35**: Network Metrics (Gas, Block Time, TPS gauges).
- **36-40**: Whale Alert Feed (Animated notification system).

### Phase 3: Web3 Intelligence (Commits 41-70)
- **41-50**: Wallet Integration (Connect Wallet, Profile Card).
- **51-60**: Portfolio Overview (Asset breakdown, Charting).
- **61-70**: Smart Contract Interaction (Transaction signing UI).

### Phase 4: Cinematic Enhancements (Commits 71-90)
- **71-80**: 3D Gas Heatmap (CSS-based or SVG-based 3D visualization).
- **81-85**: Interaction Sounds & Haptic feedback simulations.
- **86-90**: Advanced HUD Overlays (Scan-line effects, CRT flicker).

### Phase 5: Final Polish & Launch (Commits 91-100)
- **91-95**: Performance Optimization & SEO.
- **96-98**: Bug fixes & Edge cases.
- **99**: Production Build & Final Audit.
- **100**: Project Launch / Final Deployment.

## Aesthetics
- **Palette**: `Midnight Black (#050505)`, `Electric Cyan (#00F0FF)`, `Plasma Purple (#7000FF)`.
- **Effects**: Backdrop blur (20px+), thin borders (1px, 0.1 alpha), glow effects on active states.
